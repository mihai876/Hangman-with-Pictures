🎯 Hangman with Pictures – Виселица с картинками 
Классическая игра «Виселица» с красочным ASCII-артом и расширенными возможностями.
Угадывайте слова по буквам, смотрите, как рисуется виселица, и соревнуйтесь с самим собой!
Реализована на 7 языках программирования с единым интерфейсом, выбором тематик, уровнями сложности, подсказками и сохранением рекордов.

🚀 Возможности
Словарь – более 100 слов на русском языке (можно расширить).

Тематические категории – животные, города, еда, профессии, общее (переключается опцией).

Уровни сложности – количество допустимых ошибок: лёгкий (10), средний (7), сложный (5).

ASCII-виселица – 11 стадий отрисовки (от пустой до полной).

Подсказка – открыть первую букву (снимает одно очко).

Статистика – количество игр, побед, поражений, текущая серия.

Цветной вывод – угаданные буквы, ошибки и сообщения выделены.

Автосохранение – рекорды и статистика сохраняются в файл ~/.hangman_stats.json.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [режим] [опции]
Режимы
easy – 10 ошибок (по умолчанию).

medium – 7 ошибок.

hard – 5 ошибок.

Опции
Опция	Описание
-t, --theme <тема>	Тема слов: animals, cities, food, professions, general (по умолчанию general)
-s, --stats	Показать статистику и выйти
-r, --reset	Сбросить статистику
-h, --help	Справка
Управление
Введите букву (одну) и нажмите Enter.

? – подсказка (открывает первую букву, доступно один раз за игру).

quit – выход.

Примеры
bash
# Стандартная игра (легкий уровень, общая тема)
python hangman.py

# Средний уровень, тема "животные"
python hangman.py medium -t animals

# Показать статистику
python hangman.py -s

🧠 Логика игры
Выбор слова – случайное слово из выбранной темы (или общей).

Отображение – слово скрыто под прочерками, виселица показывает текущее состояние.

Ход игрока – ввод буквы. Если буква есть – открывается, если нет – добавляется ошибка и рисуется следующий этап виселицы.

Победа – когда все буквы открыты.

Поражение – когда количество ошибок достигает лимита.

Статистика – после каждой игры обновляется и сохраняется.

✨ Дополнительные фичи
Подсказка – открывает первую букву за счёт штрафа в статистике.

Цветной вывод – ANSI-коды для терминалов.

Автосохранение – рекорды хранятся в JSON-файле.
